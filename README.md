Submitted by: **Naomi Berryman**

## Summary

**CodeMath** is an android app that performs a simple calculation based on user input.  
Users can enter a number, tap a button, and instantly see the result of the math operation displayed on screen.  
This project demonstrates handling input, performing calculations, updating UI elements, and creating an interactive experience.
